# Twitter/X Research Skill

## Description
Specialized skill for analyzing real-time trends, sentiment, and discussions on Twitter/X using Puppeteer and Brave Search.

## Capabilities
- **Trend Analysis**: Identify what people are saying about a specific technology or event.
- **Sentiment Mapping**: Gauge the community's reaction to recent LLM releases or project updates.
- **Influencer Identification**: Find key opinion leaders (KOLs) in specific technical domains.

## Instructions for the Agent
1. Use **Brave Search** with the `site:twitter.com` or `site:x.com` operator to find relevant threads.
2. Use **Puppeteer** to navigate to specific thread URLs for deep reading.
3. Summarize the findings, highlighting:
    - Main consensus
    - Counter-arguments
    - Notable technical insights or "leaks"
4. Format the output as a "Social Sentiment Report."
