# Multica Knowledge Hub

Multica is the single source of truth for research context, ideas, and decisions for the AiSci project. This documentation framework establishes standards for capturing information from various sources and organizing it effectively within Multica.

## Directory Structure

- `AGENT_QUICK_START.md`: Practical guide for agents on when and how to create issues.
- `TAGGING_TAXONOMY.md`: Standardized label system for categorizing knowledge.
- `templates/`: Issue templates for different types of knowledge capture.
- `workflows/`: Source-specific guides for importing context (Perplexity, Claude, etc.).
- `guides/`: Guides for searching and discovering information in Multica.

## Core Principles

1. **Quality over Quantity**: Capture significant research, decisions, and findings.
2. **Structured but Flexible**: Use templates for consistency but adapt to the content.
3. **Linkable**: Connect related issues to build a knowledge graph.
4. **Agent-Native**: Designed to be used and maintained by both humans and AI agents.

## Getting Started

- **Agents**: Read [AGENT_QUICK_START.md](./AGENT_QUICK_START.md) to understand your role in the Knowledge Hub.
- **Humans**: Refer to the templates and taxonomy when creating or reviewing issues.
